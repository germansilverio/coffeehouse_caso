# CoffeeHouse — Caso de estudio (portfolio)

Página web navegable que cuenta cómo se construyó CoffeeHouse, de idea a producto en uso.

## Antes de publicar: completá dos links
Abrí `index.html` y buscá al final, en el `<script>`, estas dos líneas:

    const APP_URL  = "";  // poné acá la URL de tu app
    const REPO_URL = "";  // poné acá la URL de tu repo

Pegá tus URLs entre las comillas. Mientras estén vacías, los botones "Abrir la app" y "Código" se ocultan solos (la página igual funciona).

## Publicar (mismo método que la app)
Podés subir esta carpeta como un repo nuevo (ej. `coffeehouse-caso`) y activar GitHub Pages,
o crear una carpeta `/caso` dentro del repo de la app. Cualquiera de las dos sirve.
Después linkeás la URL resultante desde tu LinkedIn / CV.

## Archivos
- `index.html` — el caso de estudio (una sola página)
- `img/` — las fotos (barra antes, pantallas, instalación en la pared)
- `bitacora-del-proyecto.md` — la versión larga en texto, por si querés el detalle completo
